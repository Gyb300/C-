# C- C++ Helper Plugin

A lightweight header library that simplifies C++ syntax.

> 📌 **Open Source Statement**: Versions v1.5 and earlier are fully open source under the MIT License. Free for secondary creation.

---

## 🚀 Quick Start

```cpp
#include "C-v.1.2.h"

int main() {
    out("Hello, World!", endl);
    
    string name;
    in(name);          // input
    out("Hello, ", name, endl);
    
    return 0;
}
